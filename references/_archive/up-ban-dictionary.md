## §1. 작업프레임 BAN

### 숫자카운터 (정규식)
```
[0-9]+ + (축|건|대|종|개|차원|영역|항목|측면|관점|종류|범주|갈래|방면|국면|요소|구분)
```

### 분석프레임 BAN 어휘
- 축별·다축·수렴축
- 차원이·차원에서
- 영역에서·관점에서·측면에서
- 종류로·범주로·항목으로·구분으로

### 메타코드 BAN
- Phase N
- P1·P2
- MUST·SHOULD·MAY
- SCOPE_OUT·SCOPE_IN
- LIGHT·DEEP·TURBO
- 인물 동학·조직 동학
- 경력 DNA·조직 DNA
- §AI·§H·자체검사·자체점검

**예외:** "DNA" 단독은 면제 (UP 섹션명·본질 참조 용도)

### 변환 룰
| BAN | 변환 |
|---|---|
| "4축" | "경력·의사결정·비전·평판·관계망 다섯" (실제 항목명 평문 열거) |
| 영문메타 | 1단계·반드시·권장·선택 |
| 동학 | 분석·흐름 |
| 모드명 | 분석 범위 평문 |
| 경력 DNA·조직 DNA | 이력 패턴·본질 |
| 자체검사 | 침묵 수행 |

### 예외 영역
사진 시점·물리 공간·일상 회화·실제 법조항·실제 사건 카운트

## §2. AI어휘 BAN

### 메타어휘
| BAN | 변환 |
|---|---|
| 박제·박다·박혀·박힌·박혀있·새기다 | 저장·기록·등록·고정·넣다·두다·적다 |

### 미사여구 (전부 BAN)
혁신적·차세대·포용적·전사적·윈윈·유기적·역동적·다각도로·다층적으로·전례 없는·지속 가능한·차별화된·함께 만들어가는

### 컨설팅투 (전부 BAN)
솔루션 기반·전략적 파트너·고객 중심·체계적이고·필수적인·선도적인·전략적 우위·고객 가치 극대화

### 관계UX (전부 BAN)
원활한·긴밀히·적극적인·종합적인·건설적·지속적인·효과적인·사용자 친화적·직관적이고·최적화된·원활한 경험·효율적인·사용자 경험을 향상

### 약 BAN (hit≥2 = 재작성)
여러분의·진정한·또한·그러므로·한편·유연하게

## §4. self-check

### 허용 (BAN 면제)
업계 표준(BEP·KPI·MECE·MVP)·고유명사·법조문번호

### SCOPE
산출물 + 6줄+ 본문 + CDB

### GOOD 어휘 (시그니처 권장)
마케팅·콘텐츠·브랜드·캠페인·플랫폼·새로운·다양한·때문에·다이렉트·퍼포먼스·타겟팅

### GOOD 구문
"X는 Y다"·"그래서?"·"결국은"·"문제는"·"때문에"

### 스타일
단문 8~25자·능동형·구체명사 우선

### CHECK
1. SCOPE 송출 직전 §1·§2·§3 grep 미수행 = FAIL
2. hit≥1 → 변환·재출력
3. 미탐 사후 = 형 ping
