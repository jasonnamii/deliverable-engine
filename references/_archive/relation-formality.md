## 3. 자동추론 신호 grep

### 3-1. LV4 신호 (격식 최상)
```
키워드 grep:
투자자·VC·심사역·라운드·시리즈·IR자료·이사회·주주총회
정부·관청·부처·공공기관·국정감사·국회·청와대
고객사 임원·CEO·대표이사·회장·사장
펀드·매도·매수·M&A·실사·DD
```

### 3-2. LV3 신호 (격식 정중)
```
키워드 grep:
고객사·거래처·파트너·제안서·견적·계약서
RFP·나라장터·공공조달·입찰
외부 발신·외부 미팅·신규 고객·B2B
협력사·벤더·공급사
```

### 3-3. LV2 신호 (정중 디폴트)
```
키워드 grep:
사내 임원·타팀·상위리더·본부장·실장·팀장
주간보고·월간보고·분기보고
사내 공식 문서·전사 공유
```

### 3-4. LV1 신호 (간결)
```
키워드 grep:
내부·메모·복기·정리·기록·체크리스트
동료·팀원·우리 팀·내 메모
초안·드래프트·아이디어·브레인스토밍
형 직접·내가 볼·나만 볼
```

## 4. 자동추론 결정 트리

```
1. 요청 메시지·문서 헤더·맥락에서 §3 키워드 grep
2. hit 결과:
   - LV4 hit ≥1 → LV4 결정
   - LV3 hit ≥1 (LV4 hit=0) → LV3 결정
   - LV2 hit ≥1 (LV3·LV4 hit=0) → LV2 결정
   - LV1 hit ≥1 (다른 LV hit=0) → LV1 결정
   - 모든 hit = 0 → 디폴트 LV3 + 1회 핑
   - 다중 LV hit (예: LV4·LV1 동시) → 디폴트 LV3 + 1회 핑
3. 1회 핑 발동 시 형 응답 대기 (5초)
4. 형 응답 → LV 확정 → §B 진입
```

### 4-1. 1회 핑 표준 문구
```
🤔 수신자 확인 부탁드립니다.

이 문서, 누구에게 보내시는 건가요?
[A] 투자자·이사회·VC·정부 (격식 최상)
[B] 거래처·외부 파트너·신규 고객 (격식 정중)
[C] 사내 임원·타팀 (정중 디폴트)
[D] 동료·내부 메모 (간결)
```

## 5. LV별 종결·어조·호칭 적용

### LV4 (격식 최상) — 투자자·이사회·정부
- **종결:** 입니다체 100% (~입니다·~합니다·~됩니다)
- **어조:** 정중·간결·단정. 우회 종결 (~할 수 있을 것입니다) BAN
- **호칭:** "귀사·당사·드립니다·올립니다"
- **금지:** 친근체 (~예요·~네요), 반말 (~다·~지), 농담·환기 표현

### LV3 (격식 정중) — 거래처·외부
- **종결:** 입니다체 100%
- **어조:** 정중. 단정과 정중 균형
- **호칭:** "당사·드립니다·말씀드립니다"
- **금지:** 친근체, 반말

### LV2 (정중 디폴트) — 사내 상위·타팀
- **종결:** 입니다체 디폴트
- **어조:** 디폴트 정중. 단정형 더 자유
- **호칭:** 평문 (회사명·직책)
- **허용:** 간결한 명사형 종결, 압축형

### LV1 (간결) — 내부 메모·동료
- **종결:** 해라체 허용 (~한다·~이다)
- **어조:** 간결·압축·속도 우선
- **호칭:** 평문
- **허용:** 단정형·명사형 자유

## 6. 작성 진입 전 사전결정 (PRE_WRITE_GUARD 룰7)

### 6-1. §B 진입 직전 자기검열
> **"수신자 LV 결정됐나? 어떤 LV인가? 종결·어조·호칭 LV 매트릭스 적용했나?"**

- YES → §B 진입
- NO → §A-4 관계슬롯 재진입

### 6-2. 작성 매 문장 자기검열
> **"이 문장 종결·어조·호칭, LV에 맞나?"**

- YES → 진행
- NO → 즉시 재생성

## 7. 안티패턴 카탈로그

| 안티패턴 | LV 위반 |
|---|---|
| LV4 문서에 친근체 (~예요·~네요) | 외부 결례 |
| LV4 문서에 반말 (~다·~지) | 외부 결례 |
| LV3 문서에 농담·환기 | 외부 부적절 |
| LV1 문서에 합쇼체 강제 (~합니다 도배) | 과격식·어색 |
| 한 문서에 LV 혼재 | 일관성 붕괴 |
| LV 결정 없이 §B 진입 | 사전생성 본질 위반 |

## 8. Gotchas

| 함정 | 대응 |
|---|---|
| 수신자 신호 0인데 디폴트 무시하고 LV 추측 | 디폴트 LV3 + 1회 핑 강제 |
| 다중 LV hit 시 임의 결정 | 1회 핑 강제 |
| 1회 핑 회피 ("LV3로 가겠습니다") | 모호 = 핑 강제. 회피 = FAIL |
| LV1 = 격식 ✗로 오해 (반말 허용 ✗) | LV1 = 해라체 허용. 반말 ≠ 해라체 |
| LV 결정 후 §B 작성 중 망각 | 매 문장 자기검열 룰7 |
