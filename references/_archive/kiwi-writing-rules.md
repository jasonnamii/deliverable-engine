## 2. 5법칙 요약 (트리거 어휘 중심)

### 법칙 1. 기초편 (Syntax·구체·동사·능동·부사·구두점)
- 어순: 관련 낱말 나란히. 주어·동사 사이 절 삽입 ✗
- 강조어: 문장 끝에 배치 → 전달력 ↑
- 추상 → 구체: 색깔 → 빨간색, 개 → 은색푸들
- 동사 정곡: 부사 도움 ✗. 강한 동사로 충분
- 능동형: 수동형 회피 ("개가 고양이를 물었다")
- 부사 자제: '천천히' → 어슬렁거렸다. '매우·너무나·대단히' ✗
- 구두점: 콜론 (목록), 세미콜론 (두 주절)

### 법칙 2. 중복 불가
- '것' 남용 ✗ (예: "강조하고 싶은 점은 ... 사실이다")
- '도·등' 자주 쓰지 말라 (예: 두께가 ... 크기도 ... 글자 수 역시)
- 주어 반복 ✗ (그가 ... 그가 ... 그의 ... 그의)
- 단어·문장 중복 ✗ ("새로운 기능의 신개념 기술")
- 어미 변화 (생각했다 → 의견을 나누었다)

### 법칙 3. 금지
- 과잉 수식·수사 ✗ (수사법 = 화장 진한 얼굴)
- 이중 주어 ✗ (1문장 = 1주어)
- 자신없는 표현 ✗ (~인 것 같다·~라고 한다·~이 아닌가 싶다)
- 생뚱 단어 ✗ ("브레이크스루" → "컨설팅 회사 브레이크스루")

### 법칙 4. 축약
- 한자투 ✗ ("탑승하고 있던" → "탄"·"인기를 얻고 있는 현 상황 속에서" → "인기 속에서")
- 필요 없는 비교 ✗ ("요즘은 예전과 다르게" → "요즘")
- 과잉 감정 ✗ ("제일 먼저" → "먼저"·"것은 물론" → 삭제)
- 빼도 좋을 조사 빼라 ("이름 아래에" → "이름 아래"·"사람의 마음" → "사람 마음")

### 법칙 5. 단문 쓰기
- 문장 허리 끊기 (장문 → 단문 분절·명사형 종결 활용)
- 접속사 활용 (균형 — 남발 ✗·필요할 때 변주)

## 3. 변환 매트릭스 (BAN → CONVERT)

### 3-1. '것' 남용 BAN
| BAN | 변환 |
|---|---|
| ~하는 것이다 | ~한다·~이다 |
| ~하는 것은 | ~함은·~하기는 |
| ~한다는 것은 | ~한다는 점은·~한다는 사실은 |
| ~책이라는 것 | ~책 |

### 3-2. '도·등' 남용 BAN
| BAN | 변환 |
|---|---|
| A도 B도 C도 (3+ 연속) | A가·B도·C 역시 (어미 변주) |
| A·B·C·D 등 | A·B·C·D처럼 / 같은 이들 |

### 3-3. 자신없는 표현 BAN (단정형 강제)
| BAN | 변환 |
|---|---|
| ~인 것 같다 | ~이다 / ~인 듯하다 |
| ~인 것 같습니다 | ~입니다 |
| ~라고 한다 | ~이다 |
| ~인 것이다 | ~이다 |
| ~이 아닌가 싶다 | ~이다 |
| ~할 것인 것 같다 | ~할 것이다 / ~할 듯하다 |
| ~인지도 모른다 | ~일 수도 있다 (단, 정말 모를 때만) |
| 언젠가 (시간 모호) | 1976년·작년·구체 |

### 3-4. 한자투 BAN
| BAN | 변환 |
|---|---|
| ~로 인하여 | ~로·~때문에 |
| ~에 있어 | ~에서·~에·~로 |
| ~함에 있어 | ~할 때·~하면서 |
| ~로 인해 | ~로·~때문에 |
| 탑승하고 있던 | 탄 |
| 작용하기엔 | 되기엔 |
| 현 상황 속에서 | 지금·요즘 |

### 3-5. 부사 자제 BAN
| BAN | 변환 |
|---|---|
| 매우 / 너무나 / 대단히 / 정말 (강조 부사) | (삭제) |
| 천천히 걷는다 | 어슬렁거렸다 / 빈둥거렸다 |
| 빨리 뛰었다 | 내달렸다 / 질주했다 |
| 크게 웃었다 | 웃음을 터뜨렸다 |

### 3-6. 빼도 좋을 조사
| BAN | 변환 |
|---|---|
| 이름 아래에 | 이름 아래 |
| 사람의 마음 | 사람 마음 |
| 밤 10시 이후에 | 밤 10시 이후 |
| 진저리를 치는 | 진저리 치는 |
| 말이 없는 | 말 없는 |

## 5. 작성 토큰 단위 자기검열

### 5-1. 매 문장 직전 4문 자기검열
1. **것 남용?** 한 문장 '것' ≥ 2 → 1개로 압축
2. **도·등 남용?** '도·등' 3+ 연속 → 어미 변주
3. **자신없는 표현?** '~인 것 같다·~라고 한다' → 단정 변환
4. **한자투?** '~로 인하여·~함에 있어·탑승' → 평문 변환

### 5-2. 문단 단위 자기검열 4문
1. **부사 누적?** '매우·너무나·대단히' hit ≥ 1 → 삭제 또는 강한 동사 변환
2. **이중 주어?** 한 문장 주어 ≥ 2 → 분리
3. **장문 (25자+)?** 분절 또는 단문 분리
4. **접속사 남발?** 한 문단 접속사 ≥ 3 → 절제

## 7. 우선순위 (충돌 해소)

KIWI 5법칙은 한국 글쓰기 미학 정본. 기존 INV와 충돌 시:
- **AI 흔적 차단 (INV 16) > KIWI 5법칙** (AI식이면 KIWI 위반 여부와 무관 BAN)
- **합쇼체 종결 (INV 22) > KIWI 단정** (합쇼체 단정 = 양립 가능: "~입니다·~합니다")
- **관계격식 (INV 23) > KIWI 단정** (LV1 해라체에서만 KIWI 단정 그대로)
- **UP BAN (INV 24) > KIWI** (UP 정본이 최상위)

단순화: **UP BAN > 합쇼체 > 관계격식 > AI BAN > KIWI 5법칙 > 기타 STYLE**

## 8. Gotchas

| 함정 | 대응 |
|---|---|
| KIWI 단정형 vs 합쇼체 충돌 | 합쇼체 단정 가능 ("~입니다"). 양립 |
| KIWI '것' 남용 차단인데 면제 영역에 적용 | 면제: Headline·표·인용박스. 본문 산문만 |
| 단문 강박으로 의미 누락 | 단문은 호흡. 사실·수치 절대 보존 |
| 부사 BAN인데 형 시그니처 ('정말·확실히·진짜') 차단 | ALLOW: 시그니처 부사는 면제. 강조 부사(매우·너무나·대단히)만 BAN |
| 한자투 BAN으로 전문용어 차단 | ALLOW: 업계 전문어·법조문·고유명사 면제 |
| 접속사 균형 — 남발 vs 절제 어디서? | 한 문단 접속사 ≥ 3 = 남발. 0 = 단조로움. 1~2 = 균형 |
