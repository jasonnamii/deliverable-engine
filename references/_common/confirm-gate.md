## 목적

PRE_WRITE 자가신고 우회 차단. 자체검증 통과 = 송출 ✗ → 형 컨펌 후 송출.

## 발동

산출물 송출 *직전* 1회.

## 형식 (verbatim)


## 규칙

| 응답 | 동작 |
|---|---|
| OK | 최종 송출 |
| 수정 | 형 지적 부분만 PRE_WRITE 재진입 → 재출력 |
| 재작성 | 모드 처음부터 재실행 |

## SCOPE_OUT (컨펌 면제)
