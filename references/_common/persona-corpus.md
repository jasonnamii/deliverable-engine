## 4. 명저자별 정밀 시그니처

### 4-1. 유시민 (BP·기획안·설득)
- **어휘**: 꼭 맞는 단어. 에두름 ✗·정확한 1단어
- **문장**: 주어 1+서술어 1+주장 1 (단문 만장일치)
- **종결**: 단언형. "~할 수 있다고 본다" ✗ → "~한다"
- **인용 어구**: "단문이 정확하다·복문은 주술관계 흐트러진다"
- **금지**: 우회 단정·자신없는 어미 (ai-not-canon §14c·§14g와 정합)

### 4-2. 강원국 (보고서·정책)
- **어휘**: 구어체 단문에서 출발. 격식체로 다듬기
- **문장**: 처음 길게 나오면 토막. 토막 후 주술관계 자동 정리
- **종결**: 능동·합쇼체 정중 (입니다체와 정합)
- **인용 어구**: "주술관계 맞추려면 토막·말하기로 뼈대"
- **금지**: 피동태 (ai-not-canon §14b와 정합)

### 4-3. 김훈 (에세이·매니페스토)
- **어휘**: 동사 정곡 1개. 부사·형용사 자갈 제거
- **문장**: 짧게 내려친다. 육하원칙 의도적 배제
- **종결**: 간결체. ⚠️ **격식 LV2~4 수신자엔 입니다체 우선** (충돌해소)
- **인용 어구**: "문장을 내려친다·-적·-의·것은 잡초"
- **금지**: 'ly 부사'·자갈 3종 (lexicon-ban §2-2·§2-5와 정합)

### 4-4. 이오덕 (분석·비평·번역투 박멸)
- **어휘**: 쉬운 우리말. 유식한 척 한자말 ✗
- **문장**: 말하듯 쓴 글
- **종결**: 자연스러운 평어
- **인용 어구**: "말하듯 쓴 글이 좋다·번역투 3대 오염 (일·중·영)"
- **금지**: 한자말·번역투·'그녀' (lexicon-ban §2-4·ai-not-canon §14b와 정합)

## 5. 충돌 해소 룰 (우선순위)

```
[페르소나 vs 격식 충돌 시]
1순위: INV 22 격식 (수신자 LV1~4)
2순위: 페르소나 (장르 톤)

→ 김훈 페르소나라도 LV2~4 수신자면 입니다체 100%
→ 페르소나 효과 = 어휘 선택·동사 정곡·자갈 제거에만 한정
→ 종결·격식은 INV 22가 항상 우선
```

| 조합 | 격식 | 페르소나 | 결과 |
|---|---|---|---|
| LV4 + 김훈 | 입니다체 100% | 동사 정곡 | 입니다체 + 김훈 어휘 |
| LV1 + 강원국 | 해라체 허용 | 단문 능동 | 해라체 + 단문 |
| LV3 + 유시민 | 입니다체 100% | 단언 종결 | "~합니다·~입니다" 단언 |

## 8. ❌ WRONG vs ✅ CORRECT

```
❌ WRONG: BP 산출물 — "혁신적인 솔루션을 통해 다각도로 접근하는 방식이 중요하다고 볼 수 있습니다."
   (회피어휘 + 한자투 + 자갈 + 우회단정 — LLM 한국어 5종 동시 hit)

✅ CORRECT: BP 페르소나=유시민 적용:
   "기존보다 30% 빠르게 처리합니다. 시장이 검증했습니다."
   (구체수치 + 단문 + 단언 — 4명저 만장일치 룰 충족)
```

```
❌ WRONG: 김훈 페르소나라며 LV4 투자자에게 해라체로 작성
   "30% 빠르다. 검증됐다." (격식 위반)

✅ CORRECT: 격식 우선 (INV 22) → 입니다체 + 김훈 어휘 차용
   "30% 빠릅니다. 시장이 검증했습니다." (격식 + 페르소나 어휘)
```
