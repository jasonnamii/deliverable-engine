## §1. 형 시그니처 헤드 (단언형·≤30자)

### A. BP·IR (Cre8or Club) 결로

```
Creator 시장은 성장중
콘텐츠 산업종사자 65만 명을 넘어
시장규모 146조원
수출액 130억 달러
65만 콘텐츠 산업
3억 3백만 명
한국 인플루언서 시장
대표적인 수출품인 가전 수출액을 크게 뛰어넘은 수준
```

**패턴:** 수치 + 명사 + 단언형 결론. AI식 "혁신적 성장세를 보이며" 회피.

### B. 광고·마케팅 PT (라텔) 결로

```
외연과 내포를 통해 풀어보겠습니다
잠시 논리학에서 생각을 빌려 보았습니다
산업혁명 시대 이후의 브랜드 정의 방식
나이키도, 코크도, 펩시도 모두 '외연' 마케팅
하지만, 마케팅에서는 브랜드를 정의할 때 '내포'만을 신경 씁니다
그리고, 한정 된 개념 안에서 무엇을 해야 한다는 강박을 가집니다
```

**패턴:** 학술 인용 → 일상 사례 → 통념 깨기. 짧은 단문 + 구체 브랜드명.

## §2. 형 시그니처 구문

### A. "X는 Y다" 패턴 (코퍼스 45회 실측)

```
사업은 시작되었습니다
금요일은 특별하다
그들은 기억합니다
타입은 다르다
적지만은 않다
```

**적용:** Headline · Promise · 슬라이드 헤드 · Big Idea.

### B. "그러나·하지만" 전환 패턴

```
하지만, 마케팅에서는 브랜드를 정의할 때 '내포'만을 신경 씁니다
~옳다. 그러나 ~
```

**적용:** Twist 층 · 반대주장 박스 · CEW Warrant.

### C. 인과 명시 — "때문에" (코퍼스 96회)

```
Y가 발생하기 때문에 X가 필요하다
Z가 부족하기 때문에 W가 작동 안 한다
```

**적용:** Why Now · Complication · 진단 인과체인.
